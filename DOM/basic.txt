DOM 
---
Document Object Model
Tree structure of HTML Page 

dom.html -> run on server(Go LIVE)
1. server -> Fetches HTML  File 
2. HTML file is converted into DOM
3. css is parsed -> and then applied on DOM 
4. Js Is parsed -> then applied to DOM 
5. THE OUTPUT -> server paints that websie onto Browser screen

DOM structure -- TREE LIKE STRUCTURE
document 
|
|-- html 
|   |
|   |-- head 
|   |    |
|   |    |-- meta
|   |    |-- title
|   |    
|   |-- body
|   |    |
|   |    |-- h1
|   |    |-- p


Why do we need DOM?
1. Access HTML Elements
2. Modify HTML Content 
3. Change / Apply CSS Styles
4. Event Handling (events  , eventHandler)
5. Create or remove HTML Elements 

DOM -> Representation of HTML Page   , used by js to manipulate content on browser
