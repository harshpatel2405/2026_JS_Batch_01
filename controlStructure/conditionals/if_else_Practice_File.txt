	≥ 90% → Excellent

	0–100 units → ₹2/unit
	101–300 units → ₹3/unit

11) Input two numbers and operator (+, -, *, /):

6)  Time Slot Greeting with AM/PM	
    Input hour (1–12) and AM/PM:

		If hour ≤ 6 → "Very Early Morning"

		If hour ≤ 4 → "Good Afternoon"
				5–8 → "Good Evening"

Inputs:
	Age
	Monthly Income
	Credit Score	
Logic:
	If age ≥ 21
		If income ≥ 25,000
			If credit score ≥ 750 → “Eligible for Full Loan”
						  650–749 → “Eligible with Guarantee”
			                < 650 → “Loan Rejected: Poor Credit”
			Else → “Income Too Low”

10) Real-time Billing with Discount & Tax
	Inputs:
		Total Bill Amount
